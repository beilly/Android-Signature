# Android-Signature
Android加密小工具，基于官方命令行的工具提供签名apk的GUI工具。

![基本UI](pic/20170515115613.png)

### 特性
* 提供基于Android官方的签名功能；
* 添加UI上的签名版本选择功能；

## 环境
* eclipse-SDK-4.6.0-win32-x86_64-distro-2.4.0(自带javafx)
* JDK1.8
* JavaFX Scene Builder 2.0
